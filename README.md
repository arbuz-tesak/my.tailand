🟢 Первый блок — «Встречает нас чистый вкус Таиланда»
		Это главный экран сайта (Hero Block).
		В центре текста написано что-то вроде: “Чистый вкус Таиланда”.
		Ниже есть кнопка “Посмотреть меню”.
		При нажатии на кнопку пользователь попадает на секцию меню.
		Этот блок создаёт первое впечатление о сайте: сразу видно тему и атмосферу.
![Image Alt](https://github.com/arbuz-tesak/my.tailand/blob/main/thai-no%20idea.png?raw=true)

🟢 Меню с фильтрацией (Breakfast, Lunch, Drinks)
		Секция меню содержит маленький блок фильтров вверху:
		Started (закуски)
		Завтрак
		Ужин
		Напитки
		При нажатии на один из фильтров, например «Завтрак», пользователь видит только элементы выбранной категории.
		Это удобно, чтобы быстро ориентироваться в меню и не листать весь список блюд.																																					
		
![Image Alt](https://github.com/arbuz-tesak/my.tailand/blob/main/menu-tast-thiland.png?raw=true)


🟢 Футер (нижняя часть сайта)
		В футере есть несколько разделов:
		Меню (ссылки на разные секции сайта)
		Другие рестораны (ссылки на филиалы или партнеров)
		Контакты (телефон, email, соцсети)
		Цель футера — дать пользователю возможность легко связаться с рестораном и навигацию по сайту.
